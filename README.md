# Paper

A theme as simple as paper for Hexo

## Installation

### Install

``` bash
$ npm install hexo-renderer-inferno
$ git clone --depth 1 https://github.com/holmofy/hexo-theme-paper themes/paper
```

### Enable

Modify `theme` setting in `_config.yml` to `paper`.

``` diff
_config.yml
- theme: some-theme
+ theme: hellscape
```
